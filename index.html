Customer Support Agent (Azure RAG + Tools)

Tech: Azure OpenAI (chat + embeddings), Azure AI Search (vector index), Python, FastAPI

Built an end-to-end GenAI-powered customer support agent using Azure OpenAI and Azure AI Search. The application runs as a FastAPI service and supports retrieval-augmented generation (RAG), tool calling, basic guardrails, and multi-turn conversation memory.

The system answers customer questions by grounding responses in policy documents using vector search, and can also trigger tools for actions like case lookup and appointment scheduling. The project focuses on building a practical, production-style GenAI workflow using Azure services.

Spotify Data Analysis Project

Tech: Python, PySpark, Pandas, NumPy, Scikit-learn (Random Forest regression), Scikit-learn (Random Forest regression), Scikit-learn (Random Forest regression)

The Spotify Data Analysis using PySpark project explores and analyzes a Spotify tracks dataset to extract insights about artists, genres, and track characteristics. The analysis includes:

  • Exploratory Data Analysis (EDA): Identify top artists and genres based on popularity.
  • Correlation Analysis: Examine relationships between audio features like energy, tempo, and danceability.
  • Predictive Modeling: Use Random Forest regression to predict the energy of a track from its audio features.
  • Natural Language Processing (NLP): Analyze track names to extract keyword frequencies, generate word clouds, and perform sentiment analysis using Transformers.

The project demonstrates the use of PySpark for big data processing, machine learning for predictive modeling, and NLP techniques for text analysis, providing both statistical and visual insights.
